Bangun Masa Depan dengan MERALIA NETWORK

Dengan SDK yang mudah digunakan, dokumentasi lengkap, dan komunitas yang aktif, membangun aplikasi terdesentralisasi di SAMOYED.MEME menjadi lebih mudah dari sebelumnya.

Dokumentasi Contoh Kode
sample_contract.samo
contract TokenTransfer {
    address owner;
    
    constructor() {
        owner = msg.sender;
    }
    
    function transfer(address to, uint amount) public {
        require(msg.sender == owner, "Hanya pemilik yang dapat mentransfer");
        // Logika transfer token
        emit Transfer(msg.sender, to, amount);
    }
}
